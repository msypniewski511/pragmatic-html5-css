Efekty biblioteki jQuery

Wiekszosc programow pracujacych w jQuery wymaga podjecia trzech podstawowych krokow; zaznaczenia elementu na stronie, podlaczenia do nich odpowiedniej procedury obslugi zdarzenia oraz podania czynnosci wykonywanych w odpowiedzi na zaistnienie tego zdarzenia.

Efekty biblioteki jQuery

Czestymi zadaniami biblioteki jQuery, sa efekty wizualne, ktorych uzywa sie podobnie do pozostalych elementow javascriptu. Nalezy wiec najpierw wybrac elementy na stronie uzywajac metod jQuery. Nastepnie podac funkcje ktorej chcemy uzyc.
Ale kazda funkcja wizualna moze przyjmowac dodatkowe dwa argumenty. Czas dzialania funkcji oraz funkcje zwrotne (collback funkction).
Tempo mozna okreslic podajac czas wyrazony w milisekundach, badz przez ustalone wartosci: slow - 600, normal - 400 oraz fast - 200.

Podstawowe ukrywanie i wyswietlanie.

    Biblioteka Jquery udostepnia trzy podstawowe funkcje:
  • show() wyswietla element niezwlocznie jesli niejest podany czas animacji. Natomiast w przeciwnym wypadku tworzy animacje powoli wyswietlajaca element od jego gornego lewego naroznika do dolnego prawego.
  • hide() dziala w identyczny sposob co powyzsza ale w odwrotnym kierunku.
  • toggle() funkcja ta zmienia stan elementu naprzemienne go ukrywajac i wyswietlajac po wywolaniu.

Wygaszanie oraz rozjasnianie.

  • fadeIn() Funkcja stopniowy wyswietla element. Najpierw nadaje mu obszar na stronie poczym stopniowo wyswietla go.
  • fadeOut() Ta funkcja spowoduje ze nasz element stanie sie coraz mniej widoczny az w koncu zniknie.
  • fadeToggle() Ta funkcja laczy obie poprzednie. Jest uzywana do latwego wyswietlania elementu z uzyciem jednego przycisku.
  • fadeTo() Funkcja zmienia tylko przezroczystosc elementu. W tej funkcji musimy podac pzezroczystosc w przedziale od 0 do 1 oraz szybkosc animacji. Funkcja dziala niezaleznie od tego czy element jest widoczny czy nie.

Przesuwanie elementow

    Jezeli chcemy wiekszej ilosci ciekowych efektow, mozemy skozystac funkcji wysuwania i wsuwania elementow na strone
  • slideDown() funkcja sprawia, ze najpierw element niewidoczny jest wsuwany na strone. Najpierw pojawia sie jego gorna czesci a nastepnie reszta zawartosc strony jest przesuwana w dol, po czym jest wyswietlana jego zawartosc.
     $('#button').click(function(){
      $('#elementDocelowy').slideDown(1500)
     });
  • slideUp()To samo jak wyzej tylko w odwrotnym kierunku.
     $('#buton').click(function(){
     $('#elementDocelowy').slideUp(1500)
     });
  • slideToggle() natomiast funkcja naprzemiennie wykonuje powyzsze.
     $('#button').click(function(){
      $('elementDocelowy').slideToggle(czas)
     });

Animacje

Uwaga:Jesli chcemy animowac polozenie elementow uzywajac wlasciwosci left, right, top oraz bottm, konieczne bedzie przypisanie jego wlasciwosci CSS position wrtosci absolute lub relative.

Korzystajac z funkcji animate(), mozna animowac kazda dowolna wlasciwosc CSS akceptujaca wartosci liczbowe. Aby skozystac z funkcji nalezy jako argument przekazac literal obiektowy zawierajacy liste wlasciwosci CSS, ktore maja byc animowane, oraz ich wartosci np.:
 $('element_animowany').animate(
  {
   left: '650px',
   opacity: 0.5,
   fontSize: '24px'
  },
  1500
  );
Ostatni argument w tym przykladzie jest czasem trwania animacji.

Istnieje takze mozliwosc okreslenia docelowych wartosci na podstawie biezacych np:
 $('wartosc_animowana').click(function(){
  $(this).animate(
   {
    left: '+=50px'    },
   1000);
 });

Tempo animacji

Wszystkie funkcjie JQuery tworzace efekty wizualne pozwalaja na podanie argumentu tempa animacji. Biblioteka udostepnia dwie metody okresalnia tempa animacji (ang. easing): linear oraz swing. Metoda linear jest stalym tempem animacji, natomiast swing zmiennym.
Argument tempa animacji jest podawany po argumencjie czsu trwania animacji.

Wykonywanie operacji po zakonczeniu efektu.

Aby uzyskac kolejnosc efektow nalezy uzyc funkcji zwrotnych. Zazwyczaj funkcja taka jest przekazywana do funkcji pierwszej jako argument drugi badz trzci np:
 $('photo').fadeIn(1000, function(){
  $('caption').fadeIn(1000);
 });

Dodatkowa funkcja zwrotna jest delay(czas). Powoduje ona odczekanie okreslonego czasu przed rozpoczeciem kolejnego efektu zapisanego w kolejce np:
$('photo').fadeIn(1000).delay(10000).fadeOut(250);
