Jak dodać kod JavaScript do strony?

Posted by The Pragmatic HTML5 and CSS3 on Februar 10st, 2017 at 11:34AM

"Aside Are Not Page Sidebars!!!"
"The aside is designed to show content related to an article. It's good place to show related links, a glossary, or a pullout quote."

Osadzony wewnatrz dokumentu.

  <!doctype html>
  <html>
  <head>
    <meta charset="UTF-8">
    <title>Strona WWW</title>
    <script type="text/javascript">
       alert('Witaj, świecie!');
     </script>
  </head>
  <body>
  </body>
  </html>

Zewnętrzne pliki JavaScript

  <!doctype html>
  <html>
    <head>
      <meta charset="UTF-8">
      <title>Strona WWW</title>
      <script src="navigation.js"></script>
      <script src="slideshow.js"></script>
      <script>
         alert('Witaj, świecie!');
       </script>
    </head>
    <body>
    </body>
  </html>
